# Changelog

## [0.1.1] - 2026-05-21

### Fixed
- **資料庫膨脹問題**：交易記錄（`transactions`）改存至獨立的 `transactions.json`，不再於每個每日快照重複寫入。舊版每新增一天的快照就會複製一份完整交易歷史，日積月累導致硬碟用量線性成長。

### Migration
升級後首次儲存快照，app 會自動將交易記錄從最新日期檔案抽出，寫入 `transactions.json`，無須手動操作。

## [0.1.0] - 2026-05-19

### 初始版本
- 資產分布、資產走勢、績效分析、退休規劃、再平衡分析、持倉明細、交易紀錄七個分頁
- 時間加權報酬率（TWR）計算
- 啟動自動抓取 Yahoo Finance 報價 + 台灣銀行牌告匯率
- 五桶資產配置框架（核心 / 攻擊 / 全球分散 / 另類 / 現金防禦）
- 每日 JSON 快照，交易回溯調整歷史持倉
- 隱藏金額模式
- 頻道資訊 modal（EP17 配套工具）
